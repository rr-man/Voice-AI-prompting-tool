07Prompt Injection Defense
Protect against attempts to "gaslight" the agent into bypassing its own instructions.
Why this matters: a voice agent that takes live, unscreened caller input is a live attack surface. Callers can try to talk the agent out of its role, trick it into leaking the system prompt or stored data, or smuggle hidden instructions inside something that looks innocent — a request, a quoted document, even a string of harmless-sounding questions. None of this requires technical skill, just the right phrasing. Each toggle below adds a specific, tested defense to the prompt itself; the system-level defenses add protection outside the prompt, in how the agent is deployed.
